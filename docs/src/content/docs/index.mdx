## How it works

<CardGrid stagger>
	<Card title="Email Interception" icon="email">
		Incoming emails hit Cloudflare Email Routing, which passes them to a Worker for classification.
	</Card>
	<Card title="Smart Classification" icon="magnifier">
		Heuristic rules identify cold outreach, spam, newsletters, and legitimate email automatically.
	</Card>
	<Card title="Review Queue" icon="list-format">
		Unknown senders are held for manual review in a Chrome Extension sidebar — one click to approve or reject.
	</Card>
	<Card title="Sender Management" icon="setting">
		Build an allowlist and blocklist over time. Allowed senders bypass the queue automatically.
	</Card>
</CardGrid>
